## 第一次实战作业的记录
### pymudle库之如何将PDF表格扫描为电子版
原理就是不同的PDF可以分为两种图片行只能用光学OCR做识别，第二种是文字型应该是电子版转为PDF的类型，内在实现了某种转换算法所以可以进行逆转换
简而言之，该库认为文字性PDF内部实际上是oage列表组成，page是由一大堆BLOCK组成的，BLOCK包含该矩形block的四个边界点和text内容以及类型
通过设置表示信息可以将text进行分解，问题在于如何设置好合格的边界点，这里我使用的是准考证号，正则匹配它。每次定位到一个准考证号就认为是新的一行,之后进行匹配.
### streamlit库学到的东西
streamlit库是一个高度抽象的库,通过编写python的app.py脚本,实际通过streamlit run app.py命令可以实现前后端交互了,前端布局使用的是顺序执行,高度和宽度都是浏览器和该库自动适应的,布局根据python语句顺序渲染出来,同时streamlit库貌似好像还可以直接制定pandas或者matolptlib的画图就可以再前端展示
